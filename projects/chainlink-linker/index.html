chainlink-linker

View on GitHub

This tool is a GUI to create symlinked folders using the "ln" command.
All you need to do is select the item you want to symlink, and where to put the link. Then press the "Create Link!" button.
This was my first GUI program.

The following text comes from the project's makefile.

Build instructions

First, install necessary packages (all installations will be for Debian-based systems):
sudo apt-get install gcc libgtk-3-dev
Then run
make
to make the binary. It will appear as chainlink in the root build folder.

Installation instructions

You can run
sudo make install
to install just the binary OR run
sudo make install-desktop
to install the binary and an icon.

Uninstallation instructions

Run
sudo make uninstall
to uninstall the binary only OR
run
sudo make uninstall-desktop
to uninstall the binary and icon.
